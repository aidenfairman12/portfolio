# Portfolio

Personal portfolio site built with Next.js 14 and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Fill in before publishing

Edit the `ME`, `PROJECTS`, `SKILLS`, and `ABOUT_PARAGRAPHS` constants at
the top of `src/app/page.tsx`, then drop your resume PDF into `public/resume.pdf`.

## Deploy

Push to GitHub, import into [Vercel](https://vercel.com), deploy.
Point your custom domain at the Vercel project in **Settings → Domains**.
